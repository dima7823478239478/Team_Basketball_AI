# 🏀 ABL Lineup Optimizer — Подбор оптимальной пятерки

**Проект для ABL Moscow** (Любительская баскетбольная лига, Москва)

ИИ-система, которая на основе истории игр команды предлагает **самую атакующую**, **самую защитную** и **универсальную** стартовую пятерку для следующего матча.

## 🎯 Что делает проект

Анализирует статистику игроков из прошлых матчей и подбирает 3 варианта составов:

| Тип пятерки | Задача |
|-------------|--------|
| 🔥 **Атакующая** | Максимальная результативность (очки, передачи, 3-очковые) |
| 🛡 **Защитная** | Минимум пропущенных очков (подборы, перехваты, блокшоты) |
| ⚖️ **Универсальная** | Баланс атаки и защиты (композитный рейтинг) |

## 📊 Данные

Собираются протоколы матчей ABL Moscow:
- Игроки (позиция, рост, амплуа)
- Статистика за матч (очки, подборы, передачи, перехваты, блоки, потери, фолы)
- Плюс/минус игрока в каждой смене (если доступно)

## 🧠 Как работает

1. **Анализ истории** — рассчитываются средние показатели каждого игрока
2. **Нормализация** — приведение статистики к единой шкале (0-1)
3. **Подбор пятерок** — перебор возможных сочетаний с учетом позиций (1-5)
4. **Оптимизация** — максимизация целевой функции для каждого типа пятерки
